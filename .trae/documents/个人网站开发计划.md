# 个人网站开发计划

## 技术栈

* **Next.js 14** (App Router) - 核心框架

* **TypeScript** - 类型安全

* **Tailwind CSS** - 样式系统

* **shadcn/ui** - UI组件库

* **MDX** - 博客内容

* **Prisma + SQLite** - 数据库（打卡系统）

* **Framer Motion** - 动画效果

## 开发阶段

### Phase 1: 项目初始化

* 创建Next.js项目，配置TypeScript和ESLint

* 配置Tailwind CSS和shadcn/ui

* 设计项目目录结构

* 配置环境变量和基础配置

### Phase 2: 静态页面

* 首页（Hero、个人简介）

* 关于页面

* 联系页面

* 全局导航和页脚

* 响应式设计和暗色模式

### Phase 3: 博客系统

* 设计博客数据模型

* 配置MDX支持

* 博客列表页（分页、筛选、搜索）

* 博客详情页（MDX渲染、目录导航）

* 评论系统和阅读进度

### Phase 4: Labs展示

* Labs页面布局（卡片式）

* 项目分类和筛选

* 项目详情页

* Framer Motion动画

* GitHub集成

### Phase 5: 打卡学习平台

* 数据库Schema设计

* Prisma ORM配置

* 用户认证系统

* 打卡功能（连续天数统计）

* 学习计划管理

* 数据可视化

### Phase 6: 优化与部署

* 性能优化

* SEO优化

* 测试覆盖

* CI/CD配置

* 部署到Vercel
