## 目标与范围

搭建一个可持续维护的个人网站，需要先明确内容结构与技术边界。核心目标是：清晰表达“我是谁、我做什么、我输出什么”。

## 技术选型

这次选择 **Next.js + Tailwind CSS**，主要考虑：

- 组件化开发效率高
- SEO 与性能控制更稳定
- 可逐步扩展到 MDX 内容体系

```ts
type Stack = {
  framework: "Next.js";
  styling: "Tailwind CSS";
  content: "MDX";
};
```

## 结构规划

- 首页：介绍定位与入口
- 博客：输出内容与思考
- Labs：实验与原型
